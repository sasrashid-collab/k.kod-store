مادئاپ - کۆگای کۆدی کوردی

کۆدی سادەی فلاتتەر (Flutter)

ئەم کۆدە بۆ دروستکردنی ئەپی سادەی کوردی:

// main.dart
import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'ئەپی کوردی',
      theme: ThemeData(primarySwatch: Colors.purple),
      home: HomePage(),
    );
  }
}

class HomePage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('بەخێربێیت بۆ ئەپی کوردی')),
      body: Center(
        child: Text('سڵاو کوردستان!', style: TextStyle(fontSize: 24)),
      ),
    );
  }
}

کۆدی سادەی جاڤاسکریپت (JavaScript)

ئەم کۆدە بۆ وێبگەی کوردی:

// app.js
console.log('سڵاو کوردستان!');

function greet(name) {
  return `سڵاو ${name}!`;
}

const message = greet('کورد');
console.log(message);

// ئەم کۆدە دەردەچێت: سڵاو کورد!

کۆدی سادەی پایسۆن (Python)

ئەم کۆدە بۆ فێرکاری پایسۆن:

# app.py
print("سڵاو کوردستان!")

def greet(name):
    return f"سڵاو {name}!"

message = greet("کورد")
print(message)

# ئەم کۆدە دەردەچێت: سڵاو کورد!

کوردئاپ - ئامادەکەرەوەی ئەپ

بۆ گووگڵ پلەی (Google Play Store)

  1. حەوتەی گووگڵ دروست بکە لە play.google.com/console
  2. $25 دەبەیت (تەنها یەک جار)
  3. APK یان AAB فایلی ئەپەکەت بار بکە
  4. وێنە و وەسفی ئەپەکەت بنووسە
  5. چاوەڕوانی پەسەندکردن بە (١-٣ ڕۆژ)

بۆ ئاپ ستۆر (Apple App Store)

  1. حەوتەی ئەپڵ دروست بکە لە developer.apple.com
  2. $99 دەبەیت (ساڵانە)
  3. Xcode بەکاربێنە بۆ دروستکردنی ئەپ
  4. IPA فایلی ئەپەکەت بار بکە
  5. چاوەڕوانی پەسەندکردن بە (١-٢ هەفتە)

چی پێویستە بۆ ئەپێکی باش؟

  • ✅ ڕووکاری باش و کاریگەر
  • ✅ کاریگەری باش (بێ هەڵە)
  • ✅ ناوی باش و وەسفی ڕوون
  • ✅ وێنەی باشی ئەپەکە
  • ✅ پشتیوانی بۆ بەکارهێنەر

شافێر - فێرکاری زمانی کوردی

سڵاو

وەرگێڕان: Hello / Hi

سوپاس

وەرگێڕان: Thank you

بەخێربێیت

وەرگێڕان: Welcome

کورد

وەرگێڕان: Kurd (person from Kurdistan)

کوردستان

وەرگێڕان: Kurdistan (land of Kurds)

دەنگی کوردی (بۆ منداڵان)

  • باوک - Bawk (Dad)
  • بەراز - Beraz (Pig)
